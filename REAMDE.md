#Proyecto FULLSTAK 01

## instalcion

### servidor
-Acceder a la carpeta `server`
-Ejecutar este comando en terminal:

```shell 
 $ npm install
 ```
-para activar el servidor en desarrollo , ejecutar en termial:

```shell 
 $ npm run dev 
 ```

-para activar el servidor en produccion , ejecutar en termial:

```shell 
 $ npm run start
 ```
